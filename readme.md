# GoShip.it

Golang + Templ + HTMX (+ TailwindCSS + DaisyUI) component library that enables you to quickly develop an application using the GOTH stack, and put it in the hands of potential customers as quickly as possible.

The library contains DaisyUI components translated into Templ components that can be easily customized using both TailwindCSS and DaisyUI.

## Getting started

Install node dependencies:
`npm i -D`

Build TailwindCSS:
`make tw`

Generate component code/json, generate templates and run the server:
`make dev`
